## Using:
Create a div in your html and add class ```"Steamcard"``` . Add card css ```card.css``` and js code ```CreateCard.js```(type= "module")


in CreateCard.js
```
let Userid = "";
// Ur own steam id64
let webkey = ""
// ur own steam web api key
```
add Your steamid64 and Steam Web API key
